# Public Transparency & Audit Snapshot

**Aggregated data • Verifiable records • No personal data**

---

## What it is

A public transparency system that publishes aggregated marketplace revenue summaries, verified distribution batches, public wallets, and a changelog—so anyone can verify how the system operates.

---

## What we publish (public)

- Aggregated totals & daily summaries
- Verified distribution batches (audit view)
- Snapshot hashes per batch (verification fingerprint)
- Public wallet addresses with explorer links
- Public changelog for relevant settings

---

## What we do NOT publish

- Personal member data (PII)
- Promises of results, profit, or guarantees

---

## How verification works

- Revenue records are verified and processed into batches
- Each batch has a snapshot hash (stable fingerprint)
- Wallet addresses are public for on-chain verification
- Optional on-chain memo can reference the snapshot hash

---

## Access & security

- Read-only public pages and API (v1)
- Admin-only actions for verification and settings
- Idempotent processing and audit trails

---

## Public endpoints

- Transparency page (EN/ID)
- Read-only Public API v1 (aggregated data)
