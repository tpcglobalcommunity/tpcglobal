# Press Kit - Public Transparency System

## English Version

### Headline & Tagline
**Headline:** Public Transparency, Built In  
**Tagline:** Aggregated marketplace data. Verifiable records. No personal data.

### Short Description
We publish aggregated marketplace revenue activity, verified distribution batches, public wallets, and a changelog—so the community can verify how the system operates. This is transparency reporting only: aggregated data, no personal member information, and no promises—just verifiable records.

### Key Features
- Public transparency page (EN/ID)
- Aggregate-only data (no PII)
- Verifiable distribution batches
- Public wallets with explorer links
- Public changelog for relevant settings

### Disclaimer
Transparency reporting only. No profit, ROI, or guarantees are stated or implied.
